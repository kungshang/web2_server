系統需求
========

要執行 Internet Explorer 5 和 Internet 工具，您的系統必須滿足如下最低要求：

- 66 MHz 的 486 處理器（建議用 Pentium 處理器）

- Microsoft Windows 95 或 Windows 98：
  至少需要 16 MB 記憶體

  Microsoft Windows NT 4：
  至少需要 32 MB 記憶體

- Microsoft Windows NT(R) 4：
  必須執行 Service Pack 3（或更高）

- 最小安裝（僅瀏覽器）：
  	安裝需求：45 MB
  	執行需求：重新啟動後需要 27 MB
  典型安裝：
  	安裝需求：70 MB
  	執行需求：重新啟動後需要 55 MB
  完整安裝
  	安裝需求：111 MB
  	執行需求：重新啟動後需要 80 MB

- 滑鼠

- 數據機或 Internet 連線

- 光碟機（如果從光碟機安裝）

- 以上部份未列出需求的元件可能需要其他的系統資源。


安裝 Internet Explorer 5 的磁碟空間要求
---------------------------------------
不管選擇的安裝位置為何，Internet Explorer 5 和 Internet 工具安裝程式都要將其大部份檔案安裝到 Windows 作業系統所在的磁碟機。若要釋出硬碟空間，以備安裝 Internet Explorer 5 的磁碟空間之需，則必須釋出 Windows 作業系統所在的磁碟機上的空間。


安裝 Windows NT 4.0 Service Pack 4
-----------------------------------
安裝 Windows NT 4.0 Service Pack 4 之後，並在安裝 Internet Explorer 5 之前，您會看到訊息建議您安裝下列其中之一或兩者的 Service Pack，解決已知的千禧年問題：
  Internet Explorer 4.01 Service Pack 1
  Microsoft Data Access Components 2.0 Service Pack 1
Internet Explorer 5 會解決這些 Service Pack 的千禧年問題。


安裝
====

安裝 Internet Explorer 5 和 Internet 工具
-----------------------------------------
不管您選擇從網路安裝，還是將安裝檔案下載至硬碟，所有的安裝檔案都將被複製到您的硬碟上。這樣，您就可以在必要時快速重新安裝。如果要恢復硬碟空間，請連按兩下 [控制台] 中的 [新增/移除程式] 圖示，然後按一下 [Internet Explorer 5]。在 [Microsoft Internet Explorer 安裝程式] 對話方塊中，按一下 [進階] 按鈕，再按一下 [移除包含 Internet Explorer 安裝檔案的資料夾] 的選項。

您無法將 Internet Explorer 5 安裝到網路磁碟機或從網路磁碟機上執行 Internet Explorer 5，但是，您可以將其下載至網路磁碟機。要這樣做，您必須在您的電腦上將網路磁碟機對應為磁碟代號。Internet Explorer 5 無法被下載至 UNC 路徑。


在 Microsoft Windows NT 4.0 上執行 Internet Explorer 5
------------------------------------------------------
注意：您必須具有 Windows NT 的管理員權限才能安裝和解除安裝該程式。這包括安裝或解除安裝後首次啟動電腦的管理員權限。

安裝 Internet Explorer 5 後必須重新啟動電腦。某些附加元件在安裝完後也需要重新啟動。

至於在 Windows NT 4.0 機器的公司配置方面，請確定至少有 4 MB 的可用登錄空間，用於從 Internet Explorer 2.0 或 Internet Explorer 3.x 升級，或者至少有 2.5 MB 的可用登錄空間，用於從 Internet Explorer 4.x 升級。

如果您正在 Windows NT 4.0 上執行 Internet Explorer 5，必須將 [Temporary Internet Files] 資料夾（快取處理）的使用者權限設定為 [完全控制]。使用者必須有寫入權限才能將檔案儲存於 [Temporary Internet Files] 資料夾中。

如果您必須在安裝 Internet Explorer 5 之後重新安裝 Windows NT 4.0 Service Pack 3 (SP3)，當出現提示時您必須指定不要取代任何比安裝檔案更新的檔案。安裝完 SP3 後，請執行如下步驟更新登錄：
1. 按一下 [開始] 功能表，然後按一下 [執行]。
2. 鍵入：regsvr32 rsabase.dll
3. 按一下 [確定]。


在已安裝 Internet Explorer 5 的電腦上安裝 Windows NT 4.0 SP3 或 SP4
-------------------------------------------------------------------
將 Internet Explorer 5 安裝在您的電腦上之後，如果您重新安裝 Windows NT 4.0 Service Pack 3 或 4，則部分元件必須重新註冊。要這樣做，請執行以下步驟：
1. 連按兩下 [控制台] 的[新增/移除程式]。
2. 按 [安裝/移除] 標籤。
3. 按 [Microsoft Internet Explorer 5 和 Internet 工具]，然後按 [新增/移除] 按鈕。
4. 按 [修復 Internet Explorer]，然後按 [確定]。
5. 修復完成後，您必須重新啟動電腦。


在 Windows 2000 Beta 版上安裝
-----------------------------
此安裝為不支援的組態。安裝程式會顯示執行安裝，但僅更新瀏覽器的名稱而不更新元件。在 Windows 2000 Beta 版上您無法完整安裝 Microsoft Internet Explorer 5 元件。Windows 2000 beta 用戶會收到在下一個 Windows 2000 beta 版本中更新的 Internet Explorer 元件。


在具有雙重啟動功能的作業系統上執行 Internet Explorer 5
------------------------------------------------------
如果您在電腦上執行 Internet Explorer 5，而此電腦設定為同一分區上的兩個作業系統之間可雙重啟動（例如 Windows 95 和 Windows NT 4.0），又共享了同一個 [Program Files] 資料夾，則附加元件可能會發生問題。另外，當您要從 Internet Explorer 5 的任意安裝中移除一個附加元件，此元件將同時從兩個系統中移除，而不更新其他作業系統的已安裝元件資訊。原因是附加檔案都儲存在共享的 Program Files 資料夾中，而已安裝附加元件的資訊則儲存在其他位置。


安裝或升級 Windows 桌面更新
---------------------------
要安裝或升級 Windows 桌面更新，您必須在安裝 Internet Explorer 5 之前先安裝 Internet Explorer 4.01 SP1。 

如果已經安裝 Internet Explorer 5，並想要安裝 Windows 桌面更新，請執行以下步驟：
1. 移除 Internet Explorer 5。
2. 如果 Internet Explorer 4.01 SP1 未安裝在您的電腦上，請先安裝。
3. 安裝 [Windows 桌面更新]。
4. 重新安裝 Internet Explorer 5。


在 Windows 98 阿拉伯文和希伯來文上安裝
--------------------------------------
Windows 98 阿拉伯文和希伯來文需要相對的 Internet Explorer 5 版本。在 Windows 98 阿拉伯文或希伯來文版本上無法安裝英文版的 Internet Explorer。Windows 95 或 Windows NT 4.0 阿拉伯文或希伯來文版本則無此限制。


在 Windows Terminal Server 上安裝 Outlook Express
-------------------------------------------------
將 Outlook Express 5 安裝到 Windows Terminal Server 時，管理員必須在登錄表中設定一個相容標記，以便 Outlook Express 5 安裝程式可以正常執行。要這樣做，請執行以下步驟：
1. 建立登錄表機碼：
	HKEY_LOCAL_MACHINE\Microsoft\Windows NT\CurrentVersion\
        Terminal Server\Compatibility\Applications\setup50
2. 在新的 setup50 機碼下設定以下數值 (此數值應該是類型 REG_DWORD)：
	Flags = 0x108


Norton Protected Recycle Bin 啟動時安裝 Internet Explorer 5
-----------------------------------------------------------
建議您安裝 Internet Explorer 5 時關閉 Norton Protected Recycle Bin，並在執行安裝程式之前清理 Norton Protected Recycle Bin，確定您檢查過其內容，以免包含任何您要安全恢復的項目。Internet Explorer 5 安裝程序執行完後，您可以重新啟動此功能。

建議使用此方法是因為 Norton Protected Recycle Bin 會終止移除安裝程式試圖刪除的暫存檔，如果所剩的可用空間有限，您要使用的磁碟空間可能會不夠。這樣會導致 Internet Explorer 安裝程式被鎖定。若需關閉及重新啟動 Norton Protected Recycle Bin 的指示，請參閱 Norton Utilities 的文件。


在已加裝 Novell IPX/IP 用戶端的 Windows 98 上安裝
-------------------------------------------------
若您在 Windows 98 使用 Novell IPX/IP 用戶端，則若從 Internet 安裝 Internet Explorer 5，可能會發生問題。若安裝程式沒有回應，請重新啟動電腦。安裝程式會繼續正常完成。要解決此問題，請輸入下列其中一個選項：
- 從 Internet 選擇 [僅下載] 選項，然後正常執行和安裝
  Internet Explorer。
- 從 CD-ROM 安裝 Internet Explorer 5。


解除安裝 Internet Explorer 5 和 Internet 工具
---------------------------------------------
您可以執行以下步驟解除安裝 Internet Explorer 5 和 Internet 工具：
1. 在 [控制台] 中，連按兩下 [新增/移除程式] 圖示。
2. 按一下 [Microsoft Internet Explorer 5 和 Internet 工具]，然後按
   一下 [新增/移除]。
3. 依照螢幕上的指示執行。


無法解除安裝 Internet Explorer 4.x
----------------------------------
安裝 Internet Explorer 5 之後，將無法解除安裝 Internet Explorer 4.x。解除安裝 4.x 版的備份檔案已被移除，以便備份版本 5 的檔案。解除安裝版本 5 之後，您將無法解除安裝版本 4.x。


其他問題
========

沒有連上 Internet 時，顯示 [離線工作] 對話方塊
----------------------------------------------
如果沒有連上 Internet，而且選取 [永遠不撥號連線] 選項，當啟動 Internet Explorer 時您會看到「網頁在離線狀態時不可用」的錯誤訊息。系統會提示您再次撥接連線或選擇離線工作。

若要提示您撥接連線：
1. 在 [工具] 功能表上，按一下 [Internet 選項]。
2. 按一下 [連線] 標籤。
3. 按以下一個選項：
   網路連線不存在時撥號
   - 或 -
   永遠使用預設的連線
4. 按一下 [確定] 或 [套用]。


FTP 使用者介面和功能
--------------------
在能夠直接存取 FTP 伺服器時，Internet Explorer 5 擁有經過改進的 FTP 使用者介面。如果在鎖定 FTP 功能的 FTP 使用者介面中發現問題，您可以在 [Internet 選項] 對話方塊的 [進階] 標籤內關閉該功能。

以下是不支援的功能：
- 透過 CERN proxy 或 Web proxy 連線到 Internet
- 連線到 VAX 或 VMS FTP 伺服器
- 從個別的應用程式或服務使用瀏覽器
- 從伺服器複製到伺服器
- 按 BACKSPACE 鍵不能回到以前的位置，除非它是 HTTP 網站。與 shell 中的操作方式一樣。
- 從 FTP 伺服器拖放，但可以拖放至 FTP 伺服器
- [編輯] 功能表的 [複製] 指令和上下文功能表無法在 FTP 伺服器的項目產生作用，但您
  可以複製本機項目，然後貼到 FTP 伺服器


Outlook Express 自動選擇功能
----------------------------
自動選擇功能需要在您的電腦上安裝相應的語系支援檔案。您可以從 Internet Explorer 5 附加元件畫面取得語系支援檔案。要打開或關閉自動選擇功能，按一下 [檢視] 功能表，然後按 [自動選擇]。


字型顯示問題
------------
如果執行非遠東語言版本的 Windows NT 4.0 並安裝了多種大字型，則會超出系統使用這些字型的能力。如果遠東語言的 Web 畫面未正確顯示，請解除安裝不必要的字型，然後再試一次。Windows NT Service Pack 的功能版本包含此問題的修正檔。


在 URL 和查詢中使用非英文字元
-----------------------------
Internet Explorer 5 使用新的標準將非英文網址 (URL) 傳輸到伺服器。URL 目前使用一種名為 UTF-8 的字元編碼標準進行編碼。如果伺服器執行的是 Microsoft Internet Information Server 4 或更新的版本，URL 將依照預期的方式正常運作。許多伺服器都不能處理用 UTF-8 編碼的 URL。若要在這些伺服器的 URL 中使用非英文字元，請執行以下步驟：
1. 按一下 [工具] 功能表，然後按 [Internet 選項]。
2. 按一下 [進階] 標籤。
3. 清除 [永遠以 UTF-8 傳送 URL] 核取方塊。
URL 的查詢部份會以完全和 Internet Explorer 4 相容的形式傳輸。


Internet Explorer 4.x 相容模式
------------------------------
Internet Explorer 4.x 相容選項對於那些需要在同一台電腦不同 Internet Explorer 版本上檢視和測試其 Web 網址的 Web 研發人員非常有用。在這種模式下，您可能無法變更瀏覽器的所有設定。使用相容選項時，應該透過 [控制台] 變更 [Internet 選項]，而不是透過瀏覽器 [檢視] 功能表上的 [Internet 選項]。


Windows 電台收音機列
--------------------
Internet Explorer 5 包含一個 Windows 收音機工具列，允許順暢播放收音機。就預設值而定，啟動瀏覽器時不會顯示該工具列。要在啟動瀏覽器時顯示該工具列，請執行下列步驟：
1. 在 [工具] 功能表按一下 [Internet 選項]
2. 按一下 [進階] 標籤
3. 在 [多媒體區域] 按一下 [永遠顯示 Internet Explorer 收音機列] 核取方塊


變更功能表和對語方塊的語言
--------------------------
可能無法在所有顯示器配接卡上變更功能表和對話方塊使用的語言。如果在變更之後，看到空白功能表或對話方塊，請執行下列兩個選項：
- 變更顯示配接卡的解析度。
- 將顯示器配接卡的驅動程式升級到更新的版本。洽詢顯示器配接卡的製造商，
  取得更新之驅動程式的相關資訊。


[自動完成] 選項
---------------
若您不想讓 [自動完成] 建議特定相符的敏感資訊，您可以：
- 在 [自動完成] 下拉清單反白，然後按 DELETE，刪除特定的 [自動完成] 項目。
- 若您想刪除登入畫面的使用者名稱/密碼組合，請刪除使用者名稱，密碼會跟著一併刪除。
- 若您想清除表單或密碼資訊，請按一下 [工具] 功能表，按一下 [Internet 選項]，然後按一下 [內容] 標籤。按一下 [自動完成]，然後按一下 [清除表單] 或 [清除密碼]。


繁體中文版 INTERNET EXPLORER 5 的已知問題
=========================================

安裝日文支援檔案造成系統嚴重當機的問題已經解決
----------------------------------------------
先前的 Internet Explorer 5 曾出現以下的問題 ── 如果在 Internet Explorer 5 瀏覽日文網頁，並選擇 [檢視][編碼][日文]，就會自動安裝日文支援檔案，不過這樣會造成系統嚴重當機。現在這個問題已經解決。您無須在開始安裝日文支援之前，先行安裝 GDI 修正檔。 


Web 發佈精靈
------------
目前在 Windows 98 平台上安裝 Web 發佈精靈，會出現兩個重複的連結，一個是在 Internet Explorer 群組內的「Web 發佈精靈」，另一個是在附屬應用程式 \Internet 工具內的「Web Publish 精靈」，但這兩個連結都會連結到相同的執行檔。


應用程式相容性
==============

Outlook Express 通訊錄數位 ID 及 Outlook 98 安裝
------------------------------------------------
如果同時安裝了 Outlook 98 和 Outlook Express，通訊錄將被共享。您無法將數位簽名新增到共享的通訊錄項目中。要解決該問題，請執行以下步驟：
1. 在 [通訊錄] 中，按一下 [工具] 功能表，然後按一下 [選項]。
2. 按一下 [不共享]。
該操作將移除共享的通訊錄，並容許將數位簽名新增到所有通訊錄項目中。


Microsoft Office 2000 PowerPoint 檔案 
-------------------------------------
如果將 Internet Explorer 5 作為 Microsoft Office 2000(R) 的一部分進行安裝，然後解除安裝 Internet Explorer 5，則 Microsoft PowerPoint 可能無法開啟某些檔案。要解決該問題，請重新安裝 Internet Explorer 5。


另存為 HTML 的 Office 2000 Excel 檔案
-------------------------------------
可以在 Internet Explorer 5 中檢視 Microsoft Excel 所產生且另存為 HTML 的試算表。Internet Explorer 5 目前還不容許從瀏覽器中將試算表另存為 HTML。要編輯試算表，可使用 Excel 開啟原來的 HTML 檔案。


Visual Studio 6.0
-----------------
當您在同一台電腦上安裝了 Microsoft Visual Studio(R) 6.0 和 Internet Explorer 5，會存在一些已知問題。這些問題可以使用 Microsoft Visual Studio 6.0 SP3 來解決。如果您要執行 Visual Studio 6.0，建議當您在 http://msdn.microsoft.com/vstudio/default.asp 可以取得 SP3 的時候，就立即升級。


DirecPC 2.0
-----------
在執行 DirecPC 2.0 的電腦上安裝 Internet Explorer 5 之後，您可能會在啟動 DirecPC 軟體時看到錯誤訊息。若要解決此問題，請與 Hughes Network Systems 聯繫以便取得 DirecPC 軟體的更新版本。


BackOffice Small Business Server
--------------------------------
BackOffice Small Business Server 的使用者在安裝 Internet Explorer 5 之前，應先升級到 Small Business Server 4.5。若未升級，則 SBS 管理主控台會在每一個螢幕交替時閃動不停。調整到較高的解析度，可以改善問題但無法根本解決。


Logitech Internet 鍵盤
----------------------
某些 Logitech Internet 鍵盤隨附的 "Key Commander" 軟體，與 Internet Explorer 5 不相容。請洽詢 Logitech 客戶支援，取得更新的版本，或者搜尋他們的 Web 網站，取得詳細資訊，網址是 http://www.logitech.com。


Diamond 視訊驅動程式
--------------------
若您從 Diamond Web 網站下載 Diamond 視訊驅動程式，然後在 Internet Explorer 上安裝，則您的系統可能無法重新啟動，原因是重要的系統檔案被置換了。下面是解決此問題的兩種方法：

開始安裝這些驅動程式之前：
1. 備製 Windows\System 或 \Winnt\System32 資料夾 (系統資料夾) 的 Comctl32.dll。
2. 安裝 Diamond 視訊驅動程式。
3. 使用開機片重新啟動電腦。
4. 重新啟動之後，將複製的 Comctl32.dll 複製到系統資料夾，置換現有的檔案。
5. 取出軟碟機的磁片，然後重新啟動電腦。 

如果已經安裝驅動程式：
1. 使用開機片重新啟動電腦。
2. 在 MS-DOS 模式下，移到包含 Internet Explorer 5 檔案的資料夾。若檔案已經下載，則預設的位置是 C:\Windows Update Setup Files，或者在 CD-ROM 上選擇正確作業系統下的 IE5 資料夾。
3. 使用下列指令行，展開 Comctl32.dll 檔案： 
    Window 95 或 Windows 98:
      extract /e /l c:\temp setupw95.cab comctl32.dll
    Windows NT：
      extract /e /l c:\temp setupnt.cab comctl32.dll
4. 將展開的 Comctl32.dll 檔案從 C:\Temp 資料夾複製到 Windows\System 或 \Winnt\System32 資料夾 (系統資料夾)，置換現有的檔案。
5. 取出軟碟機的磁片，然後重新啟動電腦。 

Internet Explorer ActiveX Tree 和蹦現式功能表控制項
---------------------------------------------------
安裝 Internet Explorer 5 的時候，安裝程式會因為此版本的瀏覽器相容性
問題而移除舊版的 ActiveX Tree 控制項和蹦現式功能表控制項。使用這些控
制項的程式需要重新安裝控制項，才能恢復功能。
