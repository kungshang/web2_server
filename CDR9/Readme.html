安裝 CorelDRAW® 9 之前

執行 CorelDRAW 9   安裝之前，請確定其他應用程式已全部 關閉，包括病毒掃瞄程式以及其他可以自系統匣或「視窗」工作列上開啟的應用程式。 否則的話，不僅安裝時間會增加，並且會影響正常的安裝程序。
不同語言版本的 CorelDRAW 9 套裝軟體無法同時安裝在一個作業系統上，如果您擁有多種語言版本的 CorelDRAW 9 套裝軟體，您只能安裝一種語言的版本。

移除舊版的應用程式

Corel 建議您不要在任何先前的版本上安裝 CorelDRAW 9 應用程式 ( 例 如 CorelDRAW 5、 CorelDRAW 6、 CorelDRAW 7、 CorelDRAW 8 )。請確定您將 CorelDRAW 9 安裝至新的資料夾。 ,

為了解除先前版本的安裝，請執行「開始」功能表的「程式集｜CorelDRAW｜Setup and Notes」中的 Corel「解除安裝」程式 ( 或從「控制台」的「新增 / 移除程式」中移除「Corel 應用程式」 )。 .

如果您選擇在先前的版本上安裝 CorelDRAW 9，會備份您的使用者可修改檔案。由於未受到安裝的影響， CorelDRWA 9 中的配置檔案將不會備份 ( 例如工作空間的變更 )。

如果您選擇將 CorelDRAW 9 安裝至已裝有任何先前版本 ( 例如 CorelDRAW 5、CorelDRAW 6、 CorelDRAW 7、CorelDRAW 8 ) 的系統裡面，並決定解除安裝 CorelDRAW 9，您可能會有使用先前版本的困難。 這種情況下，請您重新安裝先前的版本。

當安裝或解除安裝這個應用程式時，請特別注意您曾經修改 過並且用原名儲存的色彩描述檔、Scripts 或樣本檔案。 為了預防您目前修改的色彩描述檔、Scripts 或樣本檔案 被移除或覆蓋，請確定在執行安裝或解除安裝這個應用程式之前，先將修改過的檔案複製到其他位置。

磁碟空間需求

安裝所需的磁碟空間會因磁碟機的設定而有所不同。 例如，在 FAT ( File Allocation Table ─ 檔案配置表 ) 格式的磁碟機上，2KB 大小的檔案在 1GB 大小的磁碟機上會佔用 16KB，而在 2GB 大小的磁碟機上則會佔用 32KB。 安裝程式的計算方式是依照檔案實際的大小而不是所需的磁碟空間。因此，當您 安裝到 FAT 格式且容量快滿之大型且經過分割的磁碟機上面時，有可能會在安裝快完成時遇到磁碟空間 不足的錯誤訊息。

安裝時，為了節省磁碟空間，有一部份的「框架」與點陣填色不會安裝。 不過他們全部都會保留在光碟上相關的資料夾中 ( \Custom\Frames 與 \Custom\Tiles )。

註冊

由於「線上註冊精靈」並不接受亞洲字元，請您在填資料時將 IME 關閉，或使用其他註冊方法註冊您的軟體。

在使用 Matrox 卡的系統上邊輯文字時，有可能會遇到一些 有關重新整理的問題。只要將系統中的驅動程式更新到最新版 ( v4.12 或以上 ) 便可以更正這些問題。

如果將效果 ( 例如漸變、輪廓圖和濾鏡 ) 應用至包含嵌入圖形的文字字串上，圖形將不會顯示在螢幕上，列印時也一樣不會顯示。

CorelDRAW 8 在 Microsoft Windows NT 和 Windows 9x 上建立與開啟檔案時，曾發現 Adobe Type1 字型寬度不一致的問題，在 CorelDRAW 9 中己更正這個問題。不過，新的計算法會產生以下的效果：

1. 在 CorelDRAW 9 ( 於任何作業系統上執行 ) 中所建立並儲存為 CorelDRAW 8 版的檔案，在 Windows NT 上執行的 CorelDRAW 8 中 開啟時，不會有任何字型寬度的差異，但在 Windows 9x 中執行的 CorelDRAW 8 中開啟時， 文字會顯得較為窄一些。
2. 在 CorelDRAW 8 ( 於 Windows 9x 中執行 ) 中所建立且在 CorelDRAW 9 ( 於 Windows 9x 中執行 ) 中開啟的檔案將會使文字顯得稍微寬一些 ( 猶如在 Windows NT 上執行的 CorelDRAW 8 中所開啟的 )。在 CorelDRAW 8 ( 於 Windows NT 中執行 ) 中所建立的檔案在任何作業系統上執行的 CorelDRAW 9 中開啟都不會有任何問題。

3. 跨平台儲存與開啟 CorelDRAW 9 的檔案都不會有任何問題。 以上所述的不一致現象僅影響檔案外觀，對於檔案中的資料都沒有任何影響。

安裝 Microsoft® Visual Basic® for Applications 需要 Windows NT 4.0 上的管理員權限。

數位簽名

Microsoft® Visual Basic® for Applications 6.0 的 新功能之一，為防止未經授權之巨集執行的「數位簽名」技術整合。

要在 Microsoft® Visual Basic® for Applications 中使用「數位簽名」，您必須己安裝 Microsoft® Internet Explorer 4.01 或更高的版本。為了您的方便，我們已將 Microsoft® Internet Explorer 5 ( IE5 ) 包含在 Microsoft® Visual Basic® for Applications 的安裝之中。如果您選擇安裝 Visual Basic for Applications 的組件，依 預設值會選取這個選項。您可以取消選取安裝程式的 IE5 部份，但是如果在您的系統中沒有 Internet Explorer 4.01 或更高的版本， 您將無法使用 Visual Basic for Applications 專案的數位簽名。

這個 IE5 的安裝程式僅安裝 Internet Explorer 5.0 瀏覽器。它會將您目前的 Internet Explorer 3 或 Internet Explorer 4 瀏覽器更新至最新的 IE5 版本。 但是，它並不會影響您其他的 Internet Explorer 應用程式，或任何 Netscape 應用程式。 因此，如果您一直使用 Netscape Navigator® 並且從未啟動 IE5，它就不會試圖變成您的預設瀏覽器。 如果您執行 Microsoft® NT 4，若要安裝 IE5，您必須已安裝 Service Pack 3 或更高的版本。

不是使用英文版 Microsoft® Internet Explorer 的使用者， 請注意下列事項。在一些情況下，適當的 IE5 語言版本無法取用，這時會以英文版取代。也就是說， 如果您使用法文版的 IE 4.01 並且安裝使用 IE5 的數位簽名，您的瀏覽器將會變成英文版。

如果您要嘗試在專案上作數位簽名，您可以使用在 C:\Program Files\Microsoft Office\Office 目 錄中的 Selfcert.exe 程式。這個程式會讓您建立一個自我簽名的認證，可以用來對您的 Visual Basic for Applications 專案 作數位簽名。

安全性問題

由於附加在 CorelDRAW 9 中的 Microsoft® Visual Basic® for Applications，對於巨集安全性的需求變得愈來愈重要。隨著 Microsoft® Visual Basic® for Applications 提供給使用者所增加的好處，它同時也需要更加強安全性的工作。

使用 CorelDRAW 9 時，不可能感染 Microsoft® Word、Excel、PowerPoint® 或 Outlook® Visual Basic for Applications 病毒。這些病毒是特別為 Microsoft® Office 環境所寫的，不會在 CorelDRAW 環境中發生作用。但是仍需 提防而且很重要的是，有些人會嘗試寫一些會感染 CorelDRAW 的 Microsoft® Visual Basic® for Applications 病毒。

只要開啟巨集病毒保護功能 ( 這是預設值 )，CorelDRAW 9 就會防止您受到 Microsoft® Visual Basic® for Applications 病毒的感染。開啟巨集病毒保護功能後，每當您收到包含巨集的文件時， 會開啟一個對話方塊並且讓您選擇是否要啟動這個巨集。如果您不確定巨集的目的、功能或他們的來源，請您關閉巨集。 關閉您不信任的巨集後，巨集將不再有任何作用。除非您執行巨集，否則巨集無法感染您的系統。您可以在您的系統上 安裝 Microsoft® Internet Explorer 4.01 或更高的版本來增強這個程序。這會啟動數位簽名功能，不僅加強保護 ，一般來說也會簡化巨集安全性的使用。若要取得更多詳細資訊， 請自您應用程式的「工具」功能表選擇「Scripts Online」 功能表項目。

如果您不確定您的安全性設定為何，或是想要改變他們，請執行下列事項：

1. 在「工具」功能表上按一下，移到「Visual Basic」然後選擇「安全性」。

2. 選擇您想要的安全性層級。高安全性只允許開啟經過數位簽名且受信任的巨集。 未經過簽名的巨集會自動關閉。中安全性則會每次出現巨集保護對話方塊，讓您不確定巨集是否安全時關閉巨集。 低安全性則會關閉所有的安全性。( 如果您擔心病毒，這個時候請不要選擇低安全性！)

另一個有關安全性的重要事項是您的防毒軟體。請執行最新版的防毒軟體，並且 經常做掃瞄的動作。也請記得安裝防毒軟體公司最新且經過簽名的檔案，以更新您的防毒軟體。
